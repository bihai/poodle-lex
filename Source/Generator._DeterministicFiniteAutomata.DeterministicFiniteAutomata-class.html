minimize(self)

source code 

Minimizes the state machine using Brzozowski's method:

  1. Reverse the DFA into an NFA
  2. Convert back to DFA
  3. Reverse the DFA into an NFA again
  4. Convert back to a DFA This method minimizes the DFA, but does not preserve information about ids.
Returns:
a DeterministicFiniteAutomata obect containing a minimized version of the DFA
